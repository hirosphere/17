16F18313 Support Information

#pragma config Usage

#pragma config <setting>=<named value>

For example:
// FEXTOSC External Oscillator mode Selection bits: EC (external clock) above 8 MHz
// Power-up default value for COSC bits: EXTOSC operating per FEXTOSC bits
// Clock Out Enable bit: CLKOUT function is disabled; I/O or oscillator function on OSC2
// Clock Switch Enable bit: Writing to NOSC and NDIV is allowed
// Fail-Safe Clock Monitor Enable: Fail-Safe Clock Monitor is enabled
#pragma config FEXTOSC = ECH, RSTOSC = EXT1X, CLKOUTEN = OFF, CSWEN = ON, FCMEN = ON

#pragma config <setting>=<literal constant>

For example:
// FEXTOSC External Oscillator mode Selection bits: EC (external clock) above 8 MHz
// Power-up default value for COSC bits: EXTOSC operating per FEXTOSC bits
// Clock Out Enable bit: CLKOUT function is disabled; I/O or oscillator function on OSC2
// Clock Switch Enable bit: Writing to NOSC and NDIV is allowed
// Fail-Safe Clock Monitor Enable: Fail-Safe Clock Monitor is enabled
#pragma config FEXTOSC = 0x7, RSTOSC = 0x7, CLKOUTEN = 0x1, CSWEN = 0x1, FCMEN = 0x1

#pragma config <register>=<literal constant>

For example:
// FEXTOSC External Oscillator mode Selection bits: EC (external clock) above 8 MHz
// Power-up default value for COSC bits: EXTOSC operating per FEXTOSC bits
// Clock Out Enable bit: CLKOUT function is disabled; I/O or oscillator function on OSC2
// Clock Switch Enable bit: Writing to NOSC and NDIV is allowed
// Fail-Safe Clock Monitor Enable: Fail-Safe Clock Monitor is enabled
#pragma config CONFIG1 = 0xFFFF

For example:
// IDLOC @ 0x8000
#pragma config IDLOC0 = 0x3FFF

HSHS (crystal oscillator) above 4 MHz
XTXT (crystal oscillator) from 100 kHz to 4 MHz
LPLP (crystal oscillator) optimized for 32.768 kHz

EXT1XEXTOSC operating per FEXTOSC bits

SOSCSOSC (31kHz)
EXT4XEXTOSC with 4x PLL, with EXTOSC operating per FEXTOSC bits

CLKOUTEN =Clock Out Enable bit
OFFCLKOUT function is disabled; I/O or oscillator function on OSC2
ONCLKOUT function is enabled; FOSC/4 clock appears at OSC2
CSWEN =Clock Switch Enable bit
ONWriting to NOSC and NDIV is allowed
OFFThe NOSC and NDIV bits cannot be changed by user software
FCMEN =Fail-Safe Clock Monitor Enable
ONFail-Safe Clock Monitor is enabled
OFFFail-Safe Clock Monitor is disabled

OFFPWRT disabled
ONPWRT enabled
